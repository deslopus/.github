# Deslopus

Open source agent discipline.

Make coding agents produce less slop.

Deslopus is an open-source initiative for improving how AI coding agents handle real software work: long, multi-step tasks where architecture, maintainability, safety, and future change matter.

We experiment with prompts, workflows, benchmarks, review practices, and agent setups that help AI produce cleaner, safer, more production-ready code.

## What We Work On

- Agent instructions that survive real codebase pressure
- Benchmarks for multi-step maintenance work
- Review practices for AI-assisted changes
- Reusable workflows for safer agent operation
- Examples of clean, production-minded agent output

## Principles

- Prefer maintainability over cleverness.
- Treat architecture, tests, security, and future change as first-class constraints.
- Make agent behavior observable and reviewable.
- Publish practical artifacts people can run, inspect, and adapt.

## Contributing

Deslopus is early. Good contributions are specific, reproducible, and honest about tradeoffs. Useful work includes prompts, workflows, benchmark tasks, review checklists, agent setup notes, examples, and critiques backed by concrete code behavior.
